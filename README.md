# Candidate Portal

A Flutter web and mobile application serving as a candidate portal for tracking the interview process, built from a real-world interview scheduling prompt. It provides candidates with an overview of the role, a way to share date availability, and resources to prepare.

## Features
- **Interview Invitation**: Read the current status and instructions from the recruiting team.
- **Availability Scheduler**: Select 5 minimum available dates for video conference interviews using an interactive calendar.
- **Preparation Resources**: Quick links to company blogs, inclusion policies, and candidate conduct expectations.
- **Accommodations Notice**: Clear directions on how to request adjustments for the interview process.

## Setup & Run
This project uses standard Flutter tooling.

1. Ensure you have Flutter installed.
2. Clone the repository and run `flutter pub get`.
3. Run the app with `flutter run`.
